# kingsroofingnc.com

Marketing website for **Kings Roofing** — a residential and commercial roofing company in Western North Carolina.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Email:** Resend
- **Deployment:** Vercel

## Pages

- `/` — Homepage
- `/about` — About the company
- `/residential` — Residential roofing services (with location subpages)
- `/commercial` — Commercial roofing services
- `/contact` — Contact form + feedback/review flows
- `/roofing-color-pickers` — Metal roofing & shingle color selectors
- `/metal-structures/*` — Metal carport kits, pole truss kits
- Various carport/pole truss style pages

## Development

```bash
bun install
bun dev
```

Open [http://localhost:3000](http://localhost:3000).
